# HexSoftwares_Design_a_Landing_Page_for_a_New_Product
UI/UX Landing Page Design for Lay's Chips Product Promotion


##🥔 Lay's Chips Landing Page Design

##📌 Project Overview

This project was created as part of a UI/UX Design Internship Task focused on designing a high-converting landing page for a new product.

The objective was to design an engaging and visually appealing landing page for Lay's Chips by applying modern UI/UX principles such as effective CTA placement, color psychology, visual hierarchy, product-focused layouts, and user-centered design practices.

The landing page showcases multiple Lay's flavor variants through dynamic color themes and interactive product selection, creating an immersive shopping experience for users.


## 🎯 Project Objectives

- Design an attractive landing page for a consumer product.
- Improve product visibility through strong visual hierarchy.
- Create an effective Call-to-Action (CTA) strategy.
- Apply color psychology to influence user engagement.
- Maintain brand consistency across different product variations.
- Enhance user experience with intuitive navigation and layout.

## 📐 UI/UX Principles Applied

### Visual Hierarchy

The design directs user attention using:

- Color contrast

### CTA Placement

The "Buy Now" button is positioned near product information to encourage quick purchasing decisions.

Proper spacing improves:

- Readability
- Content organization

All screens maintain:

- Similar layouts
- Typography standards

- User interaction patterns
